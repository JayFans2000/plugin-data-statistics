数据统计组件使用示例

1. 流量统计 (Traffic Statistics)

统一的流量统计组件，展示网站的访客和流量数据，包括历史访问统计和实时访问情况。采用卡片式设计，无需图表。

代码示例：

<div class="xhhaocom-dataStatistics-v2-traffic" data-type="weekly"></div>

效果预览：

参数说明：
  • data-type: 时间范围，可选值：daily（日）、weekly（周）、monthly（月）、quarterly（季）、yearly（年）
  • 自动展示：页面浏览量、访问次数、访客数（历史统计）和实时浏览量、实时访客（实时统计）
  • 实时数据每30秒自动更新，带有"实时"标识
  • 响应式设计，自动适配移动端

1.5. 近30分钟网站活动 (Realtime Activity)

展示最近30分钟内的网站访问活动，包括访问时间、页面路径、设备信息等。每30秒自动更新。

代码示例：

<div class="xhhaocom-dataStatistics-v2-activity"></div>

效果预览：

参数说明：
  • 自动展示最近30分钟内的访问活动
  • 显示最多20条活动记录
  • 每30秒自动更新
  • 显示信息：访问时间、页面路径、浏览器、操作系统、设备类型、国家
  • 响应式设计，自动适配移动端

使用说明

  1. 在 HTML 中引入必要的文件： 
    <link rel="stylesheet" href="/plugins/data-statistics/assets/static/css/dataStatistics.css">
<script src="/plugins/data-statistics/assets/static/js/chart.umd.min.js"></script>
<script src="/plugins/data-statistics/assets/static/js/dataStatistics.js"></script>
  2. 在需要显示统计的地方添加对应的 div 元素，并设置合适的 class 和高度。
  3. 组件会自动初始化，无需额外 JavaScript 代码。
  4. 所有组件都支持响应式设计，会自动适配移动端。
